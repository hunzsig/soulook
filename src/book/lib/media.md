media

> 调用：hmedia.method()

---

* **soundPlay**
```
播放音效
```

* **soundPlay2Player**
```
对某个玩家播放音效
```

* **soundPlay2Unit**
```
绑定单位音效
```

* **soundPlay2Loc**
```
绑定点音效
```

* **bgm**
```
播放背景音乐
如果背景音乐无法循环播放，尝试格式工厂转wav再转回mp3
```

* **bgm2Player**
```
对玩家播放背景音乐
```

* **bgmStop**
```
停止播放背景音乐
```

* **bgmStop2Player**
```
对玩家停止播放背景音乐
```
