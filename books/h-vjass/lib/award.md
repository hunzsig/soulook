award

> 调用：haward.method()

---

* **setRange**
```
设置共享范围
```

* **forUnit**
```
奖励经验\黄金\木头给单位
子方法：
    forUnitGold 奖励单位黄金
    forUnitLumber 奖励单位木头
    forUnitExp 奖励单位经验
```

* **forGroup**
```
平分奖励经验\黄金\木头给英雄组
子方法：
    forGroupGold 奖励英雄组黄金
    forGroupLumber 奖励英雄组木头
    forGroupExp 奖励英雄组经验
```

* **forPlayer**
```
平分奖励黄金\木头给玩家组
子方法：
    forPlayerGold 奖励玩家组黄金
    forPlayerLumber 奖励玩家组木头
```
