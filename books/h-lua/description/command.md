框架已经自实现了一些便利玩家的指令功能

> 输入指令的操作

```
-apm 查看APM数值指令
APM：每分钟操作的次数，又称“手速”
```

```
-apc 设定是否自动转换黄金为木头
获得黄金超过100万时，自动按照比率转换多余的部分为木头
如果超过时没有开启，会寄存下来，待开启再转换(上限1000万)
```

```
-eff 开关特效
这个命令只有在单人时有效，可关闭大部分的特效
```

```
-gg 投降指令
输入此命令的玩家立刻会失败并赶出游戏，适合自暴自弃的玩家
```

```
-random 随机选择英雄指令
只有使用hhero的构建法才支持
-repick 重新选择英雄指令
只有使用hhero的构建法才支持
```

```
+[number]
增加视距指令
-[number]
减少视距指令
```
