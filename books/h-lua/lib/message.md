message

> 调用：hmsg.method()

---

* **echo**
* **print**
```
在屏幕打印信息给所有玩家
```

* **echoToXY**
* **printToXY**
```
在屏幕(x.y)处打印信息给某玩家
```

* **echoTo**
* **printTo**
```
在屏幕(0,0)处打印信息给某玩家
```

* **delttg**
```
删除漂浮字
```

* **createttg**
```
创建漂浮字
设置during为0则永久显示
color为6位颜色代码
```

* **getTtgSize**
```
获取漂浮字大小
```

* **getTtgColor**
```
获取漂浮字颜色
```

* **setTtgMsg**
```
设置漂浮字内容
```

* **getTtgMsg**
```
获取漂浮字内容
```

* **style**
```
执行风格展示
    scale 放大
    shrink 缩小
    toggle 放大再缩小
```

* **ttg2Unit**
```
创建漂浮文字 - 在某单位头上
```

* **ttg2Loc**
```
创建漂浮文字 - 在某点上
```

* **ttgBindUnit**
```
漂浮文字 - 绑定在某单位头上，跟随移动
```
