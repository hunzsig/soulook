物品系统

---

```
物品分为 3 类
    1、永久型物品
    2、消耗型物品
    3、瞬逝型
    
    永久型物品
        代码中 type = forever 同时也是默认的物品类型
        如武器防具饰品等
        主动的效果不会叠加，并且共享冷却时间（详情请看关于主动）
        被动的叠加有别于一般的叠加（详情请看关于被动）
        *永久型物品使用后不会减少使用次数,一般主动技能不会叠加并共享冷却时间
    
    消耗型物品
        代码中 type = consume
        如药水，效能卷轴等
        恢复型功能型居多
        *消耗型物品使用后减少1次使用次数
    
    瞬逝型
        代码中 type = moment
        如掉落在地上的黄金、木头、神符等
        *瞬逝型物品获得的同时就会被使用并消失
        

关于物品合成方式
    每个英雄最大支持使用6件物品有效
    支持满背包（6+1）合成
    支持不对等数量合成，如 A+B=2C，而不一定是A+B=C，具体用法请查看 hitem 组件
    支持两单位间物品合成（最大6+6=12件） * 暂未完成
    物品存在重量，每个单位有最大负重，获得物品时，如超过负重即使存在合成关系，也会被暂时禁止合成

关于物品叠加方式
    物品默认叠加数是1
    叠加时所有的属性都会线性相加（包括属性、攻击特效、重量等）
    * 由于属性系统支持攻击特效叠加，所以要关注平衡性

关于拆分
    每个拥有物品栏的单位都会有拆分的技能
    占用位置（2,0）请注意其他技能不要占用此格
    如果目标数量大于 1 ，那么会先将其拆分成多个单品
    如果已经是单品，并且是合成物品，会拆分为各个零件
    当合成物品有多种公式合成时

注意
    当合成物品有多种公式合成时，会询问玩家以何种公式拆分
    例如A+B=C，D+E=C，会询问是分成 A+B还是D+E

```
