属性系统(自然)

```
自然属性介绍
    自然属性是魔法的一环，独立存在，不影响原有的护甲、魔抗等影响，仅呈现辅助式伤害
    自然属性一共分为：
        - fire/火
        - soil/土
        - water/水
        - ice/冰
        - wind/风
        - light/光
        - dark/暗
        - wood/木
        - thunder/雷
        - poison/毒
        - ghost/鬼
        - metal/金
        - dragon/龙

    它们同时对应各自的抵抗属性。
    每个属性在造成对应的伤害时，会获得百分比加成，无上限，而抵抗则直接减少加成比例
    如：火加成为100%，火抵抗为40%，依然加成60%
    如：火加成为0%，火抵抗却为40%，则该次火伤害只能造成60%的伤害
    * 自然属性计算在伤害增幅之后，物理/魔法暴击之前
    
    一些特殊的属性在对应某些攻击特效的时候会有特别的增益
        
        自然属性 - 火 < 攻击伤害特效 的相辅相成 >
        自然属性 - 火
            - 灼烧伤害增加
            - 爆破伤害增加
    
        自然属性 - 土 < 攻击伤害特效 的相辅相成 >
        自然属性 - 土
            - 尘迹减少魔法暴击增加
            - 沉重加重硬直减少量增加
        
        自然属性 - 水 < 攻击伤害特效 的相辅相成 >
        自然属性 - 水
            - 冻结持续时间增加
            - 寒冷持续时间增加
        
        自然属性 - 冰 < 攻击伤害特效 的相辅相成 >
        自然属性 - 冰
            - 冻结减少攻击速度增加
            - 寒冷减少移动力增加
        
        自然属性 - 风 < 攻击伤害特效 的相辅相成 >
        自然属性 - 风
            - 击飞伤害增加
            - 眩晕持续时间增加
        
        自然属性 - 光 < 攻击伤害特效 的相辅相成 >
        自然属性 - 光
            - 致盲持续时间增加
        
        自然属性 - 暗 < 攻击伤害特效 的相辅相成 >
        自然属性 - 暗
            - 腐蚀持续时间增加
            - 混乱持续时间增加
        
        自然属性 - 木 < 攻击伤害特效 的相辅相成 >
        自然属性 - 木
            - 缠绕持续时间增加
        
        自然属性 - 雷 < 攻击伤害特效 的相辅相成 >
        自然属性 - 雷
            - 闪电链伤害增加
        
        自然属性 - 毒 < 攻击伤害特效 的相辅相成 >
        自然属性 - 毒
            - 中毒伤害增加
            - 乏力持续时间增加
            
        自然属性 - 鬼 < 攻击伤害特效 的相辅相成 >
        自然属性 - 鬼
            - 霉运增加
            
        自然属性 - 金 < 攻击伤害特效 的相辅相成 >
        自然属性 - 金
            - 沉重增加
            
        自然属性 - 龙 < 攻击伤害特效 的相辅相成 >
        自然属性 - 龙
            - 爆破伤害增加
```
